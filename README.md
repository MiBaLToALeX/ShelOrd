## Features

- Extensions, Modules, Tools & Plugins
- Windows/MacOS/Linux Support
- Compiles to machine code.
- Fast
- Secure (memory safe)
- Recursive connecting
- Encrypt the data transmitted over TCP or UDP protocol.
